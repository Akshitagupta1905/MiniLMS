## Setup Instructions

### Requirements
- Node.js 18+
- Expo Go app (Android/iOS)

### Installation

1. Clone karo:
git clone <your-repo-link>
cd MiniLMS

2. Dependencies install karo:
npm install

3. App chalao:
npx expo start

4. Phone pe Expo Go se QR scan karo

## Features
- Login / Register
- Course Catalog with Search
- Bookmark Courses
- Course Detail with WebView
- Offline Banner
- Local Notifications

## Architecture
- Expo Router - File based navigation
- Zustand - State management
- Axios - API calls
- Expo SecureStore - Token storage
- AsyncStorage - App data
